# sandraelzatom.github.io
For this portfolio project, I collaborated with Gemini (AI) to implement a Spec-Driven Development workflow. The AI acted as a technical consultant, helping me bridge the gap between my initial HTML structure and the final deployment.

How I used the AI:
First, I used the AI to help me organize my project into three sprints, ensuring I had clear "receipts" for each phase of the build. When I ran into a technical hurdle with my file naming (specifically the .html.html error) and deployment issues on GitHub Pages, the AI provided step-by-step troubleshooting. It guided me through the process of renaming files, setting up the repository correctly, and "hard refreshing" my browser to see the live updates.

What I learned:
Through this collaboration, I learned that professional web development is as much about organization as it is about coding. I now better understand how GitHub Pages uses the index.html file in the root directory to host a site. While the AI provided the instructions and code snippets, I was responsible for the manual file management and the final verification of the site's responsiveness. This process gave me the confidence to manage a project from a local folder to a live global URL.
